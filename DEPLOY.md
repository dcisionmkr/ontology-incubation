# Netlifyデプロイ手順

## 1. アカウント準備

1. https://github.com にアクセス
2. dcisionmkr@gmail.com でアカウント作成
3. https://netlify.com にアクセス
4. 「Sign up with GitHub」でGitHubアカウントと連携

## 2. サイトのデプロイ（最も簡単な方法）

1. Netlifyにログイン後、「Add new site」→「Deploy manually」を選択
2. `index.html` が入ったフォルダ（`web/`）をそのままブラウザにドラッグ＆ドロップ
3. 自動でデプロイ完了

## 3. サイト名を変更する

1. デプロイ後、「Site configuration」→「Change site name」
2. `ontology-incubation` と入力して保存
3. URL: **https://ontology-incubation.netlify.app** に確定

## 4. 更新するとき

`index.html` を編集した後、同じ手順でフォルダをドラッグ＆ドロップするだけで上書き更新できます。
